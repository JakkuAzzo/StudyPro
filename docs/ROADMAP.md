# Course roadmap

StudyPro keeps each subject as a separate importable course. Shared practice engines remain course-agnostic; each course owns its content, sources, licence notes, and any specialist test format.

## 1. Great Britain car theory

The current built-in course provides an original 120-question starter bank, traffic-sign images, and 16 openly licensed hazard clips. The next content milestone is broader contributor-reviewed coverage and more UK road footage without reproducing the licensed DVSA question or clip bank.

## 2. UCAT

Add UCAT as a separate course with its current subtests, timed configuration, worked solutions, and original or appropriately licensed question banks. Check the official format immediately before implementation because test structure and timing can change.

## 3. BMAT archive

BMAT is no longer an active admissions test. If added, it should be a clearly labelled archived course for legacy practice rather than presented as a current examination. Any successor admissions-test material should be a separate current course.
